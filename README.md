## Tech Stack

- Kotlin
- Jetpack Compose
- MVVM with StateFlow
- Compose Navigation

## Architecture

- UI → Composable screens with hoisted state
- ViewModel → Handles business logic
- Data → `DummyData` as a mock repository

## Setup

1. Clone the repository
2. Open in Android Studio
3. Sync Gradle
4. Run on emulator or device
